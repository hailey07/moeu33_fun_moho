## ✨ 主要功能：

1. 兼容所有 Moho 版本（12、13、14）
2. 支持所有语言
3. 完全免费
4. 保留点运动通道中的关键帧数据、间隔和插值。
5. 如果您希望效果基于特定点做出反应，只需在应用脚本之前选择该点即可。
6. 如果您在第 0 帧变换形状，效果将在整个动画过程中跟随该变化，并保持相同的相对位置、旋转和缩放。
7. 要排除某些形状，请在应用脚本后按 Q 键，选择这些形状，然后从红色的“选定形状效果变换”通道中删除关键帧。
