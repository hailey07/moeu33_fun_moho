在 Photoshop 中使用 `MU_Name_Fix_PS.jsx` 将图层名转为 Unicode 编码；
在 Illustrator 中使用 `MU_Name_Fix_AI.jsx` 将图层名转为 Hex 编码；

再在 Moho 中使用 `MU_Name_Fix_PS.lua`  `MU_Name_Fix_AI.lua` 将代码转回原本图层名即可。
