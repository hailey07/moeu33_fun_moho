## V3.0 更新：
1. 增加了对非零帧工具操作的支持
2. 添加尖角选项
3. 新增使用Shift仅水平或垂直移动点的功能
4. 新增使用Esc在最后创建的点结束曲线的功能
5. 新增对点的网格捕捉支持
6. 添加了在绘图时禁用贝塞尔图柄激活的选项
7. 添加了设置光标距离以激活贝塞尔图柄的功能
8. 增加了点击选择点的能力
9. 新增使用Del键删除选中点的功能
10. 新增移动选中点功能
11. 增加了通过将一个点移动到另一个点或曲线上来焊接的能力
12. Enter 键现在在绘图模式下将点焊接到另一个点或曲线。
13. 现在可以在线段上移动刚刚创建的点了
14. 新增移动点时使用Ctrl键调整曲率的功能
15. 增加了移动点时使用 Alt 键解锁相邻贝塞尔图柄的功能
16. 选中一个点时可以控制贝塞尔曲线手柄
17. 添加了显示所有点或仅显示选定点的贝塞尔图柄的选项
18. 现在，要继续现有曲线，请双击现有点
19. 增加了使用 Shift + Alt 键选择多个点的功能
    1. 使用 Ctrl + Shift + Alt 键添加到选择
    2. 使用 Ctrl + Alt 键从选择中删除
20. 增加一次变换多个点的功能。
    1. 添加了 Skew 变换模式
    2. 添加扭曲变换模式
    3. 添加了通过双击重置枢轴位置的功能
    4. 增加了通过远离枢轴双击取消选择点的能力
21. 为倾斜和扭曲添加了启用/禁用变换框控制器的选项
22. 错误修复
