这个脚本非常有助于控制动画的节奏。创建过冲效果，只需要调整间距即可。

《Tween Machine》是 Davoodice 编写的一个脚本。我非常喜欢这个脚本，它帮了我很多忙。

原始脚本的链接： [https://mohoscripts.com/script/DV_TweenMachine](https://mohoscripts.com/script/DV_TweenMachine)

遗憾的是，原版程序存在一些问题，导致它无法正常运行。因此，我几乎从头开始重写了它，以解决这些问题，并能够按照我自己的想法进行开发。
