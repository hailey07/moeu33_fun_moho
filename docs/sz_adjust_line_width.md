通过加/减或乘/除调整线宽

该工具允许通过添加/减去一个值或乘以/除以提供的系数来调整线宽

它适用于整个选定层，并支持矢量图层和具有任何嵌套深度的所有类型的组层（包括开关、骨骼等）
