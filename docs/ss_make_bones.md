分裂、克隆或重组骨骼的工具

该工具用于拆分、克隆和改造骨骼

使用方法：
选择一个或多个骨骼（通常是一个），然后激活并配置该工具。默认情况下，该工具会将选定骨骼的长度划分为等长的几个骨骼，将新骨骼在链中相互作为父级，并将先前的子骨骼重新分配给新创建的集合中的最后一个骨骼。

该工具主要用于绑定和设计
但如果所选骨骼和或子级已经制作了关键帧，则应该处理动画内容（重新父级和重新定位）

![ss-make-bones-demo-01-x1200.gif](%E9%AA%A8%E9%AA%BC%E5%88%86%E6%AE%B5/ss-make-bones-demo-01-x1200.gif)

选项和功能：

- 设置件数 #所选择的骨骼应该分成（或＃克隆）
- 使用父级链接，新创建的骨头父子级关系
- 链式：新骨骼通过父级链接回基础骨骼
- 共享：新骨骼与基础骨骼共享同一个父骨骼（如果基础骨骼没有，则不共享）
- 禁用父级链接以创建没有父级的新骨骼
- 根据块数重新缩放选定的骨骼，默认的简单线性比例关系为 1/#
- 使用加权骨骼逐渐减少基于斐波那契序列的新骨骼大小，其中父级的长度等于子级 + 子子级的长度......即 (8, 5, 3) 或 (89, 55, 34, 21 )
- 使用缩放强度按骨骼大小成比例缩放骨骼强度
- 禁用重置大小以全尺寸克隆/复制所选骨骼 # 次
- 使用角度偏移创建卷曲的骨骼，或以一定角度设置的骨骼（如果未链接在父链接中）
- 使用重置第一骨骼角度，如果您还希望基础骨骼具有角度偏移
- “基础骨骼”是最初选择的骨骼，它本身可能会通过拆分/克隆过程改变大小和旋转
