> 💡省流版介绍：
>
> ​	此工具为快速切换角色 **剪影模式** 的工具。

**LF_Silhouette 脚本通过自动选择骨骼层并控制线框模式，帮助您评估角色轮廓。**

**1. 概念**

*用途*

LF_Silhouette 旨在通过显示清晰的黑色轮廓（不包含内部线条和曲线），使动画制作过程中姿势和权重更容易辨认。

- 在选定的骨骼绑定上启用黑色轮廓。
- 轮廓模式激活时，关闭线框（曲线）。
- 关闭轮廓模式时，恢复原始线框状态。
- 自动选择最相关的骨骼层。

**2. 快速工作流程**

步骤如下：

​	选择角色的任意图层（图、组等）。

​	运行 LF_Silhouette 脚本。

​	脚本会尝试查找控制骨骼层：

​	首先，查找直接控制的骨骼层。

​	如果找不到，则向上遍历层级查找骨骼层。

​	如果仍然找不到，则回退到最大的组。

​	在第 0 帧，选定图层的颜色被强制设置为黑色。

​	线框仅在之前已启用时才会关闭。

​	再次运行脚本会关闭轮廓模式并恢复之前的状态。

**3. 线框控制**

启用轮廓模式之前，脚本会检查线框模式当前是否已启用。

如果线框模式已启用，则会将其关闭以显示干净的轮廓视图。

禁用轮廓模式时：

​	如果线框模式最初已启用，则会重新启用。

​	如果线框模式之前已关闭，则会保持关闭状态（不会强制启用）。

**4. 技术细节**

在第 0 帧对图层颜色通道进行操作。

使用 fLayerColorOn 和 fLayerColor 控制图层颜色。

轮廓模式激活时，会在内部保存之前的线框状态。

在执行结束时强制刷新视口和 UI。

**5. 使用技巧**

分配一个键盘快捷键，以便在普通视图和轮廓视图之间快速切换。

在回放动画时，可以使用此功能来判断姿势的清晰度。

对于检查复杂镜头中的平衡、重量和场景调度非常有用。
