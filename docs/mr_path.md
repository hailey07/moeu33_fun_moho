在所有图层上绘制骨骼和点的运动轨迹，用于间距分析。

## 版本 1.2

增加了在浮动面板中使用脚本的功能。

## 版本 1.3 新功能：

1. 所有关于路径和目标的信息现在都存储在 Moho 项目中，让你可以在项目之间自由切换，无需每次重新指定目标。
2. 现在可以同时追踪骨骼的起点和终点。
3. 更新了"目标管理器"窗口，现在始终生成随机饱和色。
4. 为每个目标单独添加了颜色随机化按钮。Alt + 点击可随机化亮度。
5. 添加了"自动随机化颜色"选项。

## 版本 1.4 新功能：

1. 更新了目标管理器窗口，允许启用和禁用单个目标，而无需从列表中删除它们。
2. 在目标管理器窗口中，目标现在使用骨骼名称而不是 ID 来标记。
3. 错误修复。
