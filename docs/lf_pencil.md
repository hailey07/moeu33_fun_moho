## **LF_PENCIL（稳定器）- 用户手册**版本：beta 4.3

1. **概述**: LF_Pencil 是一款适用于 Moho Pro 的高级绘图工具，它将多种功能集成到一个统一的工具中：  
    - **笔触稳定：** 3 种不同模式，让您的绘画更加流畅。  
    - **压力感应：** 完全支持绘图板。  
    - **集成橡皮擦：** 无需切换工具即可擦除。  
    - **自动焊接/自动填充：** 自动连接笔画并填充封闭形状。  	
    - **附加工具：** 磁铁、曲率调节和手柄操控。  
    - **时间图表 [新增]：** 动画计时可视化和控制。  
    - **焊接交叉口：** 曲线交叉口的自动焊接。  
2. **4 种模式**使用（反斜杠）键、 **浮动按钮**或**工具栏菜单**在模式之间切换。 
    - **铅笔（绿色）：** 画笔画。  
    - **橡皮擦（蓝色）：** 擦除笔触。 
    - **编辑（品红色）：** 磁铁、曲率和把手。 
    - **时间轴（红色）：** 动画计时。  
3. **控制概要** 
   - **铅笔模式**  
      - **鼠标左键：** 绘图。  
      - **Alt + 拖动：** 调整画笔大小。  
      - **Ctrl + 单击：** 油漆桶（填充焊接交叉点）。  
      - **Ctrl + 拖动：** 创建空帧 (FBF)。  
      - **Shift + 拖动：** 套索选择。  
      - **Ctrl + Shift + 拖动：** 宽度锥度（调整任意点的粗细）。  
   - **橡皮模式**  
      - **鼠标左键：** 擦除。  
      - **Alt + 拖动：** 调整橡皮擦大小。  
      - **Ctrl + 拖动：** 修剪模式（在连接处剪断笔画）。  
      - **Shift + 拖动：** 套索删除点。  
   - **编辑模式**  
      - **鼠标左键：** 磁力（平滑移动点）。  
      - **Alt + 拖动：** 磁铁半径。  
      - **按住 Ctrl 键并拖动：** 调整曲率。  
      - **Ctrl + Shift + 拖动：** 旋转手柄。  
      - **Ctrl + Alt + Shift + 拖动：** 调整手柄纬度（半径）。  
   - **时间图表模式【新增】**  
      - **鼠标左键：** 添加标记或选择带有关键帧的点。  
      - **Alt + 鼠标左键：** 移除标记。  
      - **Shift + 拖动：** 调整弧线曲率或重新定位起点/终点。  
      - **Ctrl + 拖动：** 调整抛物线扩展范围。  
4. **全局快捷方式：**  
    - **\ : 循环模式。**  
    - **[ 和 ]** ：减小/增大半径或尺寸。  
    - **输入：** 焊接交叉（0 分）/ 创建填充（1 分）/ 关闭间隙（2 分）。  
    - **Esc：** 取消选择或清除时间图表中的所有选择。  
    - **上/下箭头：** 导航 FBF 帧。
    - **Shift + 上：** 重命名当前 FBF 帧。
