
## V2.9.2 （此版本由 Hungs59 改进）

### 1. 版本更新

- **更改**：将脚本版本从 2.7 更新到 2.9.2。
- **原因**：反映所做的重大更改和改进，确保用户可以追踪脚本的演变。

### 2. 描述更新

- **更改**：在描述中添加了更新说明：'Updated: 03/26/2025'。
- **原因**：通知用户最后更新日期，以提高透明度。

### 3. 鼠标交互增强

- **更改**：改进了鼠标交互：
    - 添加了对 Ctrl 和 Shift 键的支持，以实现更灵活的选择行为。
    - 增强了有 Ctrl 和 Alt 修饰键的单次点击选择：
        - Ctrl + 左键点击：将坐标分配到位置 1。
        - Alt + 左键点击：将坐标分配到位置 2（如果是骨骼，则为骨骼尖端）。
    - 为矢量层添加了边缘选择功能，允许用户通过点击边缘选择整个形状。
    - 将单次点击与拖动选择的鼠标距离阈值设置为 mouseDist < 8。
- **原因**：这些更改使脚本更直观，允许用户通过鼠标点击直接分配坐标，并改进对骨骼和矢量点的选择。

### 4. 尝试实现骨骼父子关系功能（未成功）

- **说明**：我尝试实现骨骼父子关系功能，使新骨骼可以成为已选择骨骼的子骨骼。然而，由于骨骼层次结构和选择逻辑的问题，我未能成功实现此功能。

### 5. 层更新修复

- **更改**：创建骨骼后，不会自动跳转到骨骼层。
- **原因**：防止意外的层焦点更改，确保更流畅的用户体验，不打断工作流程。

### 6. 本地化更新

- **更改**：将越南语本地化字符串翻译为英语以保持一致性，同时保留现有的英文字符串，未更改中文本地化块。

### 7. 界面更新一致性

### 8. 代码清理和优化

### 9. 日期格式调整

- **更改**：在 Description 函数中将日期格式从 'Cập nhật: 26/03/2025' 更改为 'Updated: 03/26/2025'。
- **原因**：与英文本地化保持一致，并遵循英文用户的标准日期格式。

## 功能改进总结

- **增强的鼠标交互**：添加了 Ctrl 和 Alt 修饰键以直接分配坐标，并改进了矢量层的边缘选择。
- **界面稳定性**：修复了层焦点问题，以获得更流畅的体验。
- **本地化**：将越南语字符串翻译为英语以保持一致性。
- **代码优化**：简化和清理代码以提高可维护性。
