## 使用方法

1. 在工具栏选择 **追踪骨骼** 工具。
2. **设置目标**：
   - 在视图中选中你希望**接收**动画的骨骼。
   - 点击工具栏上的 **"Set Target" (设置目标)** 按钮。
3. **设置源**：
   - 在视图中选中你希望**提供**动画的骨骼。
   - 点击工具栏上的 **"Set Source" (设置源)** 按钮。
4. **配置参数**：
   - 根据需要勾选要追踪的通道（位移、角度、缩放）。
   - 设置是否需要保持偏移或翻转数值。
5. **执行追踪**：
   - 点击 **"Track Bone" (追踪骨骼)** 按钮。
   - 脚本将计算并把源骨骼的动画“烘焙”到目标骨骼上。
