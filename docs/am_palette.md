该脚本支持骨骼图层通道（位置、缩放、旋转）以及图层内部的动画。这意味着您可以保存姿势预设，包括矢量动画、插值唇形同步、特殊姿势等。

## 使用方法：

1. 将骨骼图层移动到项目根目录，并使用“[变换图层（付费）](https://boosty.to/rigandmoho/posts/f5b57e1d-6364-4e3a-94c6-a2556a6a2231)”（或[重置图层变换](https://moho.moeu33.fun/scripts/MU_ResetLayerTransform)）工具将其位置重置为坐标 (0, 0, 0)。
2. 激活要为其生成图标的动作。
3. 在脚本面板中，以像素为单位设置图标宽度。例如：130。
4. 选择要生成图标的视口区域。
5. 按下“添加动作”按钮，等待过程完成。
6. 渲染完成后，脚本面板中会出现一个用于执行此操作的按钮。点击该按钮即可打开包含姿势预设按钮的面板。
7. 要从脚本中移除某个操作，请进入相应的操作并点击“移除操作”。

## 已知问题：

- 生成的图标相对于选定的视口区域存在偏移。

​	解决方法：将骨骼图层移动到项目根目录，并在第 0 帧将其位置重置为 (0, 0, 0)。

- 在 Windows 系统上，图标可能无法保存，并且操作面板打开时不会显示图像。

​	解决方法：将 Moho Custom Content 文件夹移动到系统文件夹之外（例如，桌面或文档）。将其放置在驱动器根目录通常可以解决此问题。

- 动作名称不得包含文件名中**禁止使用的字符**：
  - 星号 (*)
  - 竖线 (|)
  - 冒号 (:)
  - 双引号 (")
  - 小于号 (<)
  - 大于号 (>)
  - 问号 (?)
  - 正斜杠 (/)
  - 反斜杠 (\)
