此脚本是一个自动保存程序，它具有自动保存每 X 分钟的设置，并且也使用增量保存。

此脚本只需您设置一次，之后将无需再次操作，即可进行自动保存文件。
当您没有激活Moho窗口时，计时器不运行。
计时器仅在您执行操作时运行。
