## 安装方法：

1. 打开 Moho 软件；

2. 点击 菜单栏：`文件 - 打开自定义内容文件夹` ，或打开软件 `首选项` 打开 `自定义文件夹` 这个路径

3. 将下载的脚本解压后，按照脚本内的文件夹，分别放入 `自定义文件夹/Scripts` 中的对应文件夹内；

4. 重启 Moho 或按 `Shift+Ctrl+Alt+L` 。
