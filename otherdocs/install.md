1. 打开 Moho 软件；
2. 点击 菜单栏：`文件 - 打开自定义内容文件夹` ；
3. 将下载的脚本解压后，覆盖 `scripts` 文件夹；
4. 重启 Moho 或按 `Shift+Ctrl+Alt+L` 。
